Serverless App
on AWS Lambda

This is a serverless application created using AWS Lambda. Application retrieves content from an opensource API and relays it to the subscribed users via SNS topic based on a 24 hour event trigger. Second Lambda function configured to handle Sub/Unsub requests from API Gateway and to add or remove users from the SNS topic using respectively exposed REST endpoints

Todo_app
Relational DBs with PostgreSql

Todo application where a user is able to perform all the CRUD operations: create, retrive, update and delete on todos and todo lists. AJAX operations are incorporated where a user can update, receive, request and send data to the server without needing to reload the page making this an asynchronous web application
Tech stack: HTML, CSS, Javascript, Flask, Python, PostgreSql and Sqlalchemy

Containized Flask Api
Deployed on Kubernetes

The goal of this project was to get a thorough understanding of containization and orchestration techonolgies i.e. Docker and Kubernetes. Here I containerzed a Flask API, with three exposed endpoints, using Docker. I tested the app locally and then deployed the service to an EKS cluster with 3 worker nodes. Created a CI/CD pipeline using CodePipleline and CodeBuild which triggered upon merging in GitHub. Relevant permissions were allocated to CodeBuild to create and store container image on AWS ECR. Endpoints tested using curl

Flight Deals

Flight deals app was created using a combination of difference APIs. Using your nearest airport's IATA code, we are able to find cheap flights to any airport of your choice. We can compare flight prices to the historical low prices and if we come across a deal that is lower than the historical price, then we alert the customer by sending a message to their phone
