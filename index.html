Containized Flask Api
Deployed on Kubernetes

Built a Flask API with three exposed REST endpoints performing CRUD operations such as retrieve health check, create JWT token for a given email/password and retrieve an un-encrypted content of the JWT token. Containerized the application and tested it locally using Gunicorn. Deployed service to an EKS cluster with 3 worker nodes.Created a CI/CD pipeline using CodePipleline and CodeBuild which triggered upon merging in GitHub. Relevant permissions allocated to CodeBuild to create and store container image on AWS ECR. Endpoints tested using curl
